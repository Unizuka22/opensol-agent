# Disclaimer

OpenSol Agent is research software.

It analyzes public Solana data and creates heuristic investigation reports. It does not provide financial advice, legal advice, investment advice, or trading recommendations.

OpenSol Agent does not:

- Buy or sell assets
- Execute swaps
- Sign transactions
- Handle private keys
- Guarantee that a token, wallet, or transaction is safe or unsafe
- Prove fraud, manipulation, or intent

All conclusions must be verified independently. Public blockchain data can be incomplete, misunderstood, delayed, or unavailable depending on the RPC endpoint and parser support.
