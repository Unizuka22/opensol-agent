# Roadmap

## Phase 1: CLI

- Token mint reports
- Wallet activity reports
- Transaction explanations
- Markdown and JSON output
- Transparent risk scores
- Basic tests

## Phase 2: Agent Memory

- Local watchlists
- Saved investigation notes
- Address labels
- Diff reports for watched addresses

## Phase 3: Holder and Liquidity Analysis

- Complete holder graph analysis
- Token account owner clustering
- DEX pool discovery
- Liquidity depth and LP concentration checks
- Suspicious holder movement detection

## Phase 4: Dashboard or Desktop App

- Local web dashboard
- Investigation workspaces
- Address graph visualization
- Report history
- Exportable case files

## Phase 5: Plugin System

- Community analyzers
- Custom risk model plugins
- Data provider adapters
- Optional LLM explanation providers
